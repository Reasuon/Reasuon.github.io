对excel的格式化处理

要对excel进行美化的话这一步十分重要，但是因为EasyPoi将对excel的数据的操作进行了封装，所以我们可以在实体类中直接对数据进行格式化，而不用自己写方法。
当前版本对实体类的操作主要是通过注解来操作，例如@Excel里面的name代表列名，orderNum代表列数一样，所有对数据的格式化基本都可以通过注解来完成。
