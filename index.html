2020-02-07

  1. 直接制作一个静态页面的个人博客并部署到 http://username.github.io 上
  2. 购买一个新的域名，并将其的解析指向 http://username.github.io
  3. 制作一个动态页面，购买自己的云服务器，将动态页面的 API 接口指向云服务器，给予云服务器对该模块的 SSH 密钥，使用云服务器搭建博客后台，实现动态更新页面
