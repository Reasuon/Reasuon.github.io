MySQL5.7版本新特性

2018-11-08

语句兼容问题

不推荐直接在数据库中修改配置文件缓存，因为再重启数据库后悔重新加载，则需要重新删除。
如果安装在C盘的： 我们要进入 C:\ProgramData\MySQL\MySQL Server 5.7 此目录即可看到my.ini
